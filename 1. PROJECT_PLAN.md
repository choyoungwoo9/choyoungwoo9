# **Gitmon: 당신의 GitHub 활동으로 자라나는 프로필 펫**

## **프로젝트 목표**

**"GitHub 프로필에 나만의 동물 컬렉션을 만들어가는 새로운 경험을 제공합니다."**

Gitmon은 당신의 꾸준한 개발 활동(커밋)을 사랑스러운 동물 SVG 이미지로 시각화하여 GitHub 프로필에 전시하는 자동화 서비스입니다. 매일 당신의 활동을 체크하여, 새로운 동물을 컬렉션에 추가하거나 활동이 뜸할 경우 동물이 사라지기도 합니다.

이 프로젝트는 단순한 이미지 표시를 넘어, 개발자들에게 자신의 기여 기록을 가꾸고 공유하는 작은 즐거움과 동기를 부여하는 것을 목표로 합니다.

## **핵심 기능 및 사용자 경험**

*   **자동화된 프로필 업데이트:** 사용자가 Gitmon 웹사이트에 등록하고 권한을 허용하면, 서비스가 매일 자정마다 사용자의 GitHub 활동을 자동으로 분석합니다.
*   **활동 기반 동물 컬렉션:** 최근 10일간의 커밋 활동에 따라 새로운 동물이 사용자의 컬렉션에 추가되거나, 기존 동물이 사라집니다. 최대 10마리까지 다양한 동물을 수집할 수 있습니다.
*   **동적 SVG 이미지 생성:** 서버는 사용자의 최신 동물 컬렉션 데이터를 기반으로, 모든 동물이 울타리 안에 예쁘게 배치된 하나의 SVG(`gitmon.svg`) 이미지를 생성합니다.
*   **GitHub 리포지토리 직접 커밋:** 생성된 `gitmon.svg` 파일은 사용자의 GitHub 프로필 리포지토리에 자동으로 커밋 및 푸시됩니다. 사용자의 프로필에는 항상 최신 상태의 동물 컬렉션이 보여집니다.
*   **컬렉션 대시보드:** 사용자는 Gitmon 웹사이트에서 현재까지 수집한 동물들의 목록과 상태를 한눈에 확인할 수 있습니다.

## **기대 효과**

*   **개발 동기 부여 및 게임화:** 매일 새로운 동물을 수집하거나, 기존 동물을 잃지 않으려는 소소한 목표가 생겨 꾸준한 커밋을 유도합니다. 지루할 수 있는 개발 활동을 '오늘의 컬렉션'을 모으는 재미있는 게임처럼 만듭니다.
*   **꾸준함의 시각적 증거:** 풍성한 동물 컬렉션은 개발자의 최근 활동량과 꾸준함을 한눈에 보여주는 시각적인 '증표'가 됩니다.
*   **개성있는 프로필 표현:** 밋밋할 수 있는 GitHub 프로필을 자신만의 스토리가 담긴 공간으로 만들어, 방문자에게 깊은 인상을 남깁니다.
*   **소통의 시작점:** 독특한 프로필 요소는 다른 개발자들의 흥미를 유발하고, "이 귀여운 동물들은 뭔가요?" 와 같은 질문을 통해 자연스러운 네트워킹의 시작점이 될 수 있습니다.
